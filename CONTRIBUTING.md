# Contributing to t403

This is a personal experimental workspace for MetaForge, KickLang, and Google AI Edge Gallery integration.

## How to Contribute (Internal / Personal)

- All changes should be made via feature branches.
- Use clear, TAS-aligned commit messages when possible.
- New Agent Skills must pass Google AI Edge Gallery constraint validation (see `skills/featured/` examples).
- Updates to MetaForge specs should maintain both human-readable and `.kl.md` editions.

## Skill Development Guidelines

- Follow kebab-case folder naming
- Maintain strict `SKILL.md` frontmatter
- Prefer offline-first, privacy-respecting designs
- Validate JS skills against the `ai_edge_gallery_get_result` contract

## Questions

Open an issue or discuss in the relevant KickLang / edge-gallery-forge sessions.
